## Audio Setup for Demo Recording (CRITICAL)

### Option A: BlackHole (Best — captures both voices)
```bash
# Step 1: Install
brew install blackhole-2ch
# Enter sudo password when prompted
# REBOOT after install

# Step 2: Create Multi-Output Device
# Open: /Applications/Utilities/Audio MIDI Setup.app
# Click "+" at bottom-left → "Create Multi-Output Device"
# Check both:
#   ☑ Built-in Output (MacBook Pro Speakers)
#   ☑ BlackHole 2ch
# Set "Built-in Output" as Master Device (clock source)
# Rename to "Demo Recording" (optional)

# Step 3: Set as system output
# System Settings → Sound → Output → select "Multi-Output Device"

# Step 4: In screen recording (Cmd+Shift+5)
# Options → Microphone → select "BlackHole 2ch"
# This captures: VibeCat voice + your voice + all system audio
```

### Option B: Without BlackHole (Fallback — your voice only)
```
Cmd+Shift+5 → Options → Microphone → MacBook Pro Microphone
# VibeCat's spoken audio will play through speakers and be picked up by mic
# Quality will be lower but workable
# Speak clearly, stay close to mic, reduce background noise
```
