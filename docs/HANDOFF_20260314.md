## Demo Recording Procedure

### Pre-recording setup:
```bash
# 1. Install BlackHole (if not done)
brew install blackhole-2ch
# Reboot, then open Audio MIDI Setup → create Multi-Output Device

# 2. Build and run VibeCat
cd VibeCat && swift build
# Launch VibeCat — it should auto-connect to production gateway

# 3. Verify connection
curl -s https://realtime-gateway-163070481841.asia-northeast3.run.app/health
# Should show connections: 1

# 4. Open required apps:
# - Antigravity IDE (with a Swift file open)
# - Terminal
# - Chrome with YouTube Music tab
```

### Recording:
```
Cmd+Shift+5 → Screen Recording → Options → Audio: Multi-Output Device
Follow docs/DEMO_VIDEO_SCRIPT.md exactly
Keep under 4 minutes
```

### Post-recording (agent can help):
```bash
# Trim and compress
ffmpeg -i raw_recording.mov -t 240 -c:v libx264 -crf 23 -c:a aac -b:a 128k demo.mp4

# Add subtitles (agent can generate .srt file from script)
ffmpeg -i demo.mp4 -vf subtitles=subtitles.srt -c:a copy demo_with_subs.mp4
```
