IMPORTANT DECISIONS
-------------------
- YouTube Music은 navigate_open_url로 검색 URL 직접 열기 (type_and_submit 사용 금지)
- 재생은 비전 기반 마우스 클릭으로 (JS hack이 아닌 스크린샷→좌표→마우스이동→클릭)
- Gemini 환각 문제(도구 안 부르고 "했다"고 말함)는 inject-text의 한계. 실제 데모는 사용자 음성 입력으로 작동
- startup greeting은 "제안만, 실행 안 함" (SUGGEST only, do NOT execute)
- open_url/focus_app/submit_enter는 ProofLevel none + continue_next_step (검증 실패로 재시도하면 탭 중복)
- executeJavaScriptInChrome에 "front window" 실패 시 "window 1" fallback
- Chrome 종료는 quit + "나가기" Enter 처리
